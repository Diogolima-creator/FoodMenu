## Instalação

Instale meu projeto usando gitBash e com npm

```bash
  git clone https://github.com/Diogolima-creator/FoodMenu.git
  npm install
  npm start
```

    
## Deploy

Você pode acessar o projeto também por este link: **https://food-menu-delta.vercel.app**
